## Getting Started

```bash
git clone https://github.com/HaDock404/ai-nlp-tweet_sentiment.git
cd ai-nlp-tweet_sentiment
pip install -r ./packages/requirements.txt
```

## Launch Application

```bash
python production/application/main.py
```
